# 🎮 GameStore - Milestone 2

Gaming e-commerce site built with HTML, CSS, JS, PHP, MySQL.

## File Structure

```
gaming-store/
├── index.html              # Home page
├── products.html           # Product listing + search
├── product-detail.html     # Product detail + add to cart
├── cart.html               # Cart: modify qty, delete, buy
├── contact.html            # Contact us + Google Map
├── admin/
│   ├── login.html          # Admin login
│   ├── dashboard.html      # View / search / delete products
│   ├── add-product.html    # Add product + image upload
│   └── edit-product.html   # Edit product + image upload
├── includes/
│   └── config.php          # DB connection (PHP)
└── assets/
    ├── css/
    │   ├── style.css       # Customer styles
    │   └── admin.css       # Admin panel styles
    └── js/
        └── main.js         # Cart logic
```

## Rubric Coverage

| Requirement | File |
|---|---|
| List products with images | `index.html`, `products.html` |
| Product detail (price, qty) | `product-detail.html` |
| Cart: delete, modify, buy | `cart.html` |
| Admin login | `admin/login.html` |
| Admin add product + image | `admin/add-product.html` |
| Admin modify product + image | `admin/edit-product.html` |
| Admin delete + search | `admin/dashboard.html` |
| Admin logout | `admin/dashboard.html` (logout link) |
| Contact us + map | `contact.html` |

## Setup
1. Import DB schema into MySQL
2. Edit `includes/config.php` with your credentials
3. Run on XAMPP/WAMP (PHP required for backend)
